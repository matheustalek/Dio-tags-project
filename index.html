h1-h6

Utilizada para a criação de textos, quanto menor o número da tag, maior a importância recebida.
Seu uso é semântico (necessária para PCD) buscadores como google buscam as tags de maior importância para organizar seus resultados de pesquisa.

p

Utilizada para criação de parágrafos, geralmente textos que demandam maior espaço na página, geralmente utilizada em artigos, notícias.

mark

Quando inserida, cria uma marcação amarela no que estiver entre as tags de abertura e fechamento

small

Deixa o que estiver entre as tags menor

i

Deixa o que estiver entre as tags em itálico

u

Deixa o que estiver entre as tags sublinhado

strong

Deixa o que estiver entre as tags em negrito
